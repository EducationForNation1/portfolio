Project information

  • Category: Web Development
  • Client: Chetu India Pvt Ltd.
  • Project URL: Cooper General

Project Description

Cooper General is a comprehensive mobile repair warehouse management system designed to streamline the internal processes of a mobile repair centre. The application enables employees to efficiently manage the lifecycle of mobile repairs, from intake to delivery, ensuring timely service and enhanced customer satisfaction.

Key Features: Repair Tracking: Employees can log and monitor the status of repairs in real-time, ensuring transparency and accountability at each stage of the process.

  • Inventory Management: Integrated tools to track spare parts and resources required for repairs, minimizing downtime due to stock shortages.
  • Customer Notifications: Automated updates are sent to customers, keeping them informed about the status of their device repairs.
  • Delivery Scheduling: Streamlined scheduling system to ensure on-time delivery of repaired devices to customers.

    Technologies Used:
  • Front-End: Django
  • Back-End: Python (Django)
  • Database: SQL for efficient data storage and management
  • API Integration: Developed RESTful APIs to connect various modules and enhance system functionality
  • Deployment: Hosted on AWS.
    My Role and Contribution
  • Developed and optimized back-end services using Django to handle repair tracking and inventory management
  • Designed and implemented RESTful APIs to ensure seamless communication between frontend and back-end modules
  • Integrated real-time repair status updates and notification systems for enhanced customer communication.
  • Collaborated with cross-functional teams to identify and resolve technical challenges, improving application performance.
  • Ensured code quality and maintainability through unit testing, debugging, and adherence to coding standards.
